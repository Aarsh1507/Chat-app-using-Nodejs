In this tutorial, we will use the Node.js platform to build a real-time chat application that sends and shows messages to a recipient instantly without any page refresh. Also, we have some validations like if the username is taken then we will have to choose some other username. If two users are in the same room then only they can see the message of each other

In this tutorial, I will show how I built my news app using React JS. So what is special in this 3-page application? You can take any API and fetch data using Axios and render it in view. I am writing a blog on this because I am using RSS Feed of ABP News and Aaj Tak news, where I am getting data in XML format. Seems interesting right? Let's get started.

You will need Node 6+ installed on your machine. Some knowledge of JavaScript and React will be helpful.

Prerequisites

To follow along, a basic knowledge of JavaScript (ES6) and React is required. You also need to have the following installed on your machine:

  • Node.js (v6 and above)
  • npm

Getting started

Start by creating a new empty directory news-app and run npm init -y from within it in order to initialize the project with a package.json file.

Set up the React app

We will bootstrap our React application with create-react-app. Install it globally on your computer by running the following command:

npm install -g create-react-app

Once the installation process is done, you can run the command below within the news-app directory to set up your React application:

create-react-app NewsApp

Once the command has finished running, cd into the newly created NewsApp folder and install the other dependencies we will be needing to build the app frontend.

So we will be creating mainly 3 components

  1. NewsCategory: It will display different types of categories in news app such as world news, sports, business, etc.

2. NewsList: It will display a list of all news from the specified category.

3. NewsDetail: It will display detailed news from the list when the user clicks on any news.

So let's go one by one through all the components

1. NewsCategory

We will use the class component as we want to manage different categories in this component, and when users click on any category that should be passed down as props to the NewsList component. So we will manage our category as an object of key-value pair in the state.

First, we are looping through all categories. Then we are passing down selected categories in Route itself as a state object.
